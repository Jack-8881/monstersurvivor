Monster Survivors

Survive endless waves of monsters in this thrilling action-packed adventure!

Game Features

  • • Endless survival gameplay
  • • Multiple character upgrades
  • • Dynamic monster waves
  • • Power-up collection system

How to Play

  • • Use WASD or arrow keys to move
  • • Collect experience orbs to level up
  • • Choose upgrades wisely
  • • Survive as long as possible

About Monster Survivors

Monster Survivors is an intense survival action game that puts your skills to the test. Face increasingly challenging waves of monsters while collecting power-ups and upgrading your character. With its roguelike elements and permanent progression system, each run offers a unique experience. Can you survive long enough to become the ultimate monster slayer?
